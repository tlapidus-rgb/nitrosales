## Reglas para usar este banco

1. No usarlos literales: son base de trabajo, ajustar a la vertical / momento.
2. Headline siempre debe poder leerse en voz alta sin tropiezos.
3. Subhead no repite el headline con sinónimos.
4. CTA es siempre un verbo concreto.
