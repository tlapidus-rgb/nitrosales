# brand-voice

Aplica o revisa la voz de NitroSales sobre cualquier texto. Esta skill es complementaria a `positioning-canon` — el canon aporta QUÉ decir, esta skill aporta CÓMO decirlo.

## Cuándo se dispara

- "Escribime esto con voz NitroSales."
- "Este texto suena genérico, ajustalo."
- "Revisá que no suene a IA."
- Al generar cualquier output comercial: landing, email, post, mensaje, slide.
- Al recibir un texto escrito por un tercero (agencia, freelancer, socio) para validar si está en voz.

## Proceso

1. Cargá el canon de voz: `/sessions/practical-vibrant-shannon/mnt/nitrosales/SKILLS_VM/_CANON/brand-voice.md`.
2. Si el texto hace claims sobre producto o features, cargá también el canon de posicionamiento.

### Para aplicar voz a un texto nuevo

1. Escribí un draft siguiendo los patrones del canon (4 patrones de estructura).
2. **Pass 1 — palabras prohibidas**: escanea el output y reemplazá cada aparición de palabras en la lista prohibida.
3. **Pass 2 — ritmo**: mezclá frases cortas (5-12 palabras) con medianas (15-25). Usá tres beats donde aplique.
4. **Pass 3 — lectura en voz alta**: mentalmente, leelo. Si suena a IA o a comunicado genérico, reescribí.
5. Devolvelo.

### Para revisar un texto existente

1. Identificá las 3-5 frases más fuera de voz.
2. Citá cada una + explicá por qué no está en voz + proponé alternativa.
3. No reescribas el texto completo salvo pedido explícito.

## Atributos de voz (checklist mental al escribir)

- [ ] **Directa**: empezá por la conclusión. Sin "en este mundo competitivo...".
- [ ] **Con alma**: hablá como si NitroSales fuera un cerebro vivo. Usá verbos de percepción.
- [ ] **Argentina sin parodia**: rioplatense natural ("vas a", "sabés"). Sin "che boludo".
- [ ] **Segura sin arrogancia**: afirmamos, no nos comparamos agresivo.
- [ ] **Honesta con los números**: si no hay data, no la inventes.

## Palabras prohibidas (flag si aparecen)

- all-in-one
- plataforma integral / solución integral
- dashboard inteligente
- revolucionar / disruptivo / cambio de paradigma
- unlocking / empowering / elevating
- boost / skyrocket / 10x
- simple / intuitivo / fácil
- herramienta (usá "capa", "cerebro", "sistema")

## Palabras preferidas

- percibe, entiende, recuerda, interpreta, piensa
- sistema nervioso, cerebro, órgano, capa
- verdad, señal, ruido
- activo digital, memoria viviente
- dólar MEP, IVA, AFIP, inflación

## Patrones de estructura (los 4)

1. **Afirmación + reforzador**: "Tu primer activo digital. No otro SaaS más."
2. **Pregunta + respuesta corta**: "¿Cuánto creciste? ¿De verdad, descontando inflación?"
3. **Dolor + verdad + CTA implícito**: "Abrís 8 pestañas. Cerrás 7 con NitroSales."
4. **Enumeración en tercios**: "Vendé más. Gastá mejor. Decidí con la verdad."

## Regla de oro: el test en voz alta

Si el texto, leído en voz alta, suena a IA o a marketing corporativo traducido → NO está en voz. Reescribí.

Ver `references/ejemplos-si-no.md` para ejemplos comentados.
