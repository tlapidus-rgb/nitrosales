# positioning-canon

Esta skill carga en contexto el **canon de posicionamiento NitroSales v5** y lo usa como marco de referencia para cualquier output comercial.

## Cuándo se dispara

- Antes de escribir landing, deck, email, post, whatsapp, ad copy, o cualquier material comercial de NitroSales.
- Cuando Tomy pregunta por el posicionamiento (one-liner, arquetipo, pilares, diferenciadores).
- Como revisión de texto existente ("¿este copy está alineado?").
- Como briefing rápido a un tercero (agencia, freelancer, socio).

## Proceso

1. Leé el archivo canon completo: `/sessions/practical-vibrant-shannon/mnt/nitrosales/SKILLS_VM/_CANON/positioning-canon.md`.
2. Leé también: `/sessions/practical-vibrant-shannon/mnt/nitrosales/SKILLS_VM/_CANON/brand-voice.md`.
3. Si el pedido incluye features o activos específicos, cargá además:
   - `/sessions/practical-vibrant-shannon/mnt/nitrosales/CONOCIMIENTO_PRODUCTO/QUE_ES_CADA_PRODUCTO.md`
   - `/sessions/practical-vibrant-shannon/mnt/nitrosales/CONOCIMIENTO_PRODUCTO/SECCIONES_DEL_PRODUCTO.md`

## Output según contexto

### Si Tomy pregunta "cuál es nuestro posicionamiento"

Respondé con **la tarjeta canónica**:

```
One-liner:     Tu primer activo digital. Vendé más. Gastá mejor. Decidí con la verdad.
Arquetipo:     Sistema nervioso / cerebro vivo.
4 pilares:     Percepción · Cognición · Memoria viviente · Verdad.
4 activos:     NitroPixel · Aurum · Bondly · Aura.
Target ICP:    Fundadores ecommerce LATAM, GMV USD 50k-1M/mes, VTEX + ML + Meta + Google.
Dolor raíz:    "Vendo bien, pero no sé si gano plata ni de dónde vienen las ventas."
```

### Si Tomy pide revisar un texto

Sacá los 3-5 puntos específicos donde el texto **no está alineado** al canon. Citá literal y proponé alternativa. No reescribas el texto completo sin pedir permiso.

### Si Tomy pide generar copy

Produciendo el output, respetá obligatoriamente:
- **Palabras prohibidas**: all-in-one, revolucionar, boost, skyrocket, simple, intuitivo, unlocking, empowering, elevating, dashboard inteligente, plataforma integral.
- **Palabras preferidas**: percibe, entiende, recuerda, sistema nervioso, cerebro, capa, órgano, verdad, activo digital, memoria viviente.
- **Ritmo**: mezcla frases cortas (5-12) con medianas (15-25). Tres beats en afirmaciones.
- **Ejemplo canon**: "Tu primer activo digital. Vendé más. Gastá mejor. Decidí con la verdad." (4/3/3/4 beats).

## Qué evitar

- No inventes nuevos pilares, no renombres los activos, no cambies el arquetipo sin pedir permiso a Tomy.
- No uses frases que podrían aplicar a cualquier SaaS. Si el copy podría ser de Triple Whale, Segment o Shopify, reescribilo.
- No omitas el pilar de Verdad si estás tocando temas de atribución, data, o contra-Meta/Google.

## Anti-patrones comunes

- **Traducción literal de copy USA**: "revolutionary" → "revolucionario". NO. Encontrá el equivalente rioplatense con alma.
- **Feature listing seco**: "tenemos atribución, P&L, creators y BI". Escondé el feature detrás del dolor: "vas a saber de qué canal viene cada venta, si estás ganando plata, y qué creator te mueve la aguja".
- **Hero genérico**: "La plataforma X para Y". NUNCA. Arrancá con dolor o afirmación específica.

## Cuándo pedirle permiso a Tomy

- Si Tomy pide cambiar el one-liner → antes de generar alternativas, confirmar que quiere iterar el canon (puede ser solo para un uso puntual).
- Si el canon y la petición chocan (ej: Tomy pide escribir con palabras prohibidas a propósito para un ejercicio) → aclarar.
